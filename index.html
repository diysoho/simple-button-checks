Simple Bt Checks

Download Issue Follow @joelthorner

Simple bt checks is a simple plugin for transform checkbox inputs to html buttons for css customize. High performance, keyboard support and preserve original input click/change events.

Getting started

1. Include Simple bt checks files to website/app

After that we need to include CSS and JS files to our website/app. In your html file:

Optionaly we can use Bootstrap

<!DOCTYPE html>
<html lang="en">
<head>
   ...
   <link rel="stylesheet" href="path/to/bootstrap3.x.x.min.css"><!-- optional framework -->
   <link rel="stylesheet" href="path/to/simple-bt-checks.min.css">
</head>
<body>
   ...
   <script src="path/to/jquery2/3.x.x.min.js"></script>
   <script src="path/to/bootstrap3.x.x.min.js"></script><!-- optional framework -->
   <script src="path/to/simple-bt-checks.min.js"></script>
</body>
</html>

2. Add HTML inputs

This plugin contemplates several label-input formats:

<!-- sibiling label-input -->
<div class="checkbox">
   <input type="checkbox" name="aValue" id="a" value="344">
   <label for="a">Address</label>
</div>

<div class="checkbox">
   <label for="b">Address</label>
   <input type="checkbox" name="bValue" id="b" value="1">
</div>

<!-- Bootstrap recomended -->
<div class="checkbox">
   <label>
      <input type="checkbox" checked="checked"> Check me out
   </label>
</div>

3. Initialize plugin

After load jquery and simple-bt-checks js files

$(document).ready(function () {
   // initialize plugin when document ready  
   $('input[type="checkbox"]').simpleBtChecks({
      size : "default",
      btnClass: "btn btn-default",
      checkedIcon : "glyphicon glyphicon-ok"
   });
});

Api

Parameters

List of all available parameters:

Parameter Type Default Description
size string 'default' Could be "default", "x-2" or "x-3"
btnClass string 'btn btn-default' Class attribute value for new html button tag
checkedIcon string 'glyphicon glyphicon-ok' icon font class (or other class)
bootstrapUse boolean true If not use Bootstrap framework set to false
wrapContainer string 'none' Values: 'none' or 'input' or 'all'. Wrap with a div label and input, only input or nothing.
strictLabel boolean|null true Add event listener if label has relation: label 'for' attr -> input 'id' attr. With false value, search sibiling or parent label. If is true and not find relationship set option to false.
With null value not find any label to set events;
btnAttributes object { type : 'button' } Add attribute name - value into new html button. Example: 
btnAttributes: {
   title : 'string value'
}
Callbacks
onInit(simpleBtChecks) function null Callback function, will be executed right after plugin initialization
onChange(simpleBtChecks) function null Function, will be executed right before checkbox input change
changeCallback(simpleBtChecks) function null Callback function, will be executed right after checkbox input change
onDestroy(simpleBtChecks) function null Callback function, will be executed when you destroy Swiper

Methods

After we initialize input/s we have its initialized instance in variable (like myInput variable in example below) with helpful methods:

var myInput = $('input[type="checkbox"]').simpleBtChecks({
   size : 'x-3'
});

myInput.simpleBtChecks('destroy');
Method Description
myInput.simpleBtChecks('destroy'); Destroy simpleBtCheck instance and detach all events listeners
myInput.simpleBtChecks('addListener', node); Add change checkbox input event to a node. Example: 
myInput.simpleBtChecks('addListener', $('#div-1'));
